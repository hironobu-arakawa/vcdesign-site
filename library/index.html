Repository Index

Authoritative repositories under VCDesign / BOA. This index defines what is inside the system.

  • vcdesign-core Core definition of VCDesign (immutable rules / concepts)
  • boa-core Boundary-Oriented Architecture: construction method under VCDesign

  • vcdesign-origins Primary sources and origins (no interpretation added)
  • interface-determinability-gate A responsibility-bound guardrail and boundary pattern for AI that must remain silent when determinability breaks.
  • factory-data-storage-notes Storage responsibility and irreversibility notes
  • factory-data-shapes Data shape catalog across the flow
  • db-layer-design-principles Responsibility principles for DB layers
  • ot-it-async-design-appendix Async-first OT/IT design patterns
  • edge-semantics-template Semantic handling template at the edge
