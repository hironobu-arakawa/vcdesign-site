Library

English note:
This Library is an index of satellite repositories in the VCDesign ecosystem. It is not meant to be read sequentially. It exists to fix boundaries and to show where each responsibility lives. Most content remains in Japanese intentionally, to preserve the nuance of “responsibility”, “Genba (field operations)”, and boundary failures that cannot be translated without distortion.

この Library は、VCDesign エコシステムの「衛星リポジトリ」を 境界ごとに固定するための索引です。 読む場所ではなく、参照先を決める場所です。
